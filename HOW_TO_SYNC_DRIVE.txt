Why the previous script stopped at 50:
Google Drive uses "Virtual Scrolling," which hides images that are not actively visible. The previous scrolling code failed to scroll the correct hidden Google Drive scrollbar container.

We have a brand new script that acts exactly like a human pressing the "Page Down" key, guaranteeing that it loads all 400 photos.

Note: DO NOT WORRY about any red errors in the console like "service worker", "DOCS_initialLoadTiming", or "CORS policy". Those are background Google Drive errors that are completely harmless and normal!

### AUTO-SCROLLER SCRIPT (Guaranteed to Work)
1. Open your Google Drive Folder link in Chrome: 
   https://drive.google.com/drive/folders/1ckzwzMFuWd2kIVX1xKnpx-PKRzVD8aMv?usp=drive_link
2. Refresh the page so you are at the VERY TOP.
3. Press **F12** on your keyboard to open Developer Tools, and click the "**Console**" tab.
4. Copy and paste the script below into the white console box and hit **Enter**:

let allItems = {};
let previousCount = 0;
let unchangedCount = 0;

let scroller = setInterval(() => {
    let items = document.querySelectorAll('div[data-id]');
    
    items.forEach(el => {
        let id = el.getAttribute('data-id');
        let text = (el.innerText + " " + (el.getAttribute('aria-label') || "")).toUpperCase();
        let match = text.match(/[a-zA-Z0-9_-]+\.(?:JPG|JPEG|PNG)/);
        if (match) {
            allItems[match[0]] = id;
        }
    });
    
    if (Object.keys(allItems).length === previousCount) {
        unchangedCount++;
    } else {
        unchangedCount = 0;
        console.log("📸 Scanned " + Object.keys(allItems).length + " photos so far...");
    }
    previousCount = Object.keys(allItems).length;
    
    // Automatically Scroll like a human focusing the last visible item
    if (items.length > 0) {
        items[items.length - 1].scrollIntoView();
    }
    
    // If it hasn't found any new photos for 8 seconds, it assumes it reached the bottom
    if (unchangedCount > 4) {
        clearInterval(scroller);
        console.log("✅ DONE! Total photos collected: " + Object.keys(allItems).length);
        let a = document.createElement('a');
        a.href = URL.createObjectURL(new Blob([JSON.stringify(allItems, null, 2)], {type: 'application/json'}));
        a.download = 'drive_links.json';
        a.click();
    }
}, 2000);

5. SIT BACK AND WATCH! You will actually see it jumping down item by item!
6. Once it reaches the bottom and stops finding new images, `drive_links.json` will automatically download containing all 400 files!
7. Drag and drop that `drive_links.json` file into this `D:\Vaishu\app\backend\` folder.
